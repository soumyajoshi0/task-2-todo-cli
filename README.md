# To-Do List CLI - Task 2

This project is a simple console based to-do list application built in Python for internship Task 2.

## Features
- Add tasks
- View tasks
- Remove tasks
- Tasks saved in a text file

## Concepts Used
- Lists
- File handling using open()
- Loops and conditionals
- User input

## How to Run
1. Open terminal in project folder
2. Run python todo.py
3. Follow menu instructions

This task helped me understand file handling and list operations in Python.
